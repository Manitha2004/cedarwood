# Cedar Woods Accommodation System

JavaFX application developed for the Object-Oriented Systems Development group assignment.

## Features
- View accommodation details by area
- View breakfast and cleaning statistics
- Guest check-in
- Guest check-out
- Cleaning status updates

## Technologies Used
- Java
- JavaFX
- NetBeans
- IntelliJ

## Team Project
This repository contains the Java source files for the Cedar Woods Accommodation System group project.
